## Pairwise Swap in a Linked List
### Difficulty: _`Medium`_
---
### **Statement**
Given a singly linked list, swap every two adjacent nodes.


---
### **Example 1**
- Input: `List: 1 -> 2 -> 3 -> 4 -> 5` 
- Output: `2 -> 1 -> 4 -> 3 -> 5`

### **Example 2**
- Input: `List: 8 -> 7 -> 5 -> 2 -> 1`
- Output: `7 -> 8 -> 2 -> 5 -> 1`
