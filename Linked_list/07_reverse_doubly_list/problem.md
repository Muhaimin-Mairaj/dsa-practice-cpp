## Reverse a Doubly Linked List
### Difficulty: _`Easy`_
---
### **Statement**
Given the `head` of doubly linked list, reverse the list in-place so that the first node becomes the last, the second node becomes the second last, and so on. 



---
### **Example 1**
- Input: `List: 1 <-> 2 <-> 3` 
- Output: `3 <-> 2 <-> 1`

### **Example 2**
- Input: `List: 1`
- Output: `1`
