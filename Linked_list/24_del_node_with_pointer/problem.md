## Given Only a Pointer/Reference To a Node To Be Deleted In a Singly Linked List
### Difficulty: _`Medium`_
---
### **Statement**
Given a pointer to a node to be deleted, delete the node.<br>
**Note:** We don't have a pointer to a head node.

---
### **Example 1**
- Input: `List: 10 -> 20 -> 4 -> 30, Pointer = head->next`
- Output: `10 -> 4 -> 30`

### **Example 2**
- Input: `List: 1 -> 2, Pointer = head`
- Output: `2`
