## Find First Node of a Loop in Linked List
### Difficulty: _`Medium`_
---
### **Statement**
Given the head of a linked list, determine the starting node of a loop if cycle exists. A loop occurs when the last node points back to an earlier node in the list. If no loop is present, return -1.

---
### **Example 1**
- Input: `List: 1 -> 3 -> 2 -> 4 -> 5 -> back to 3`
- Output: `3`

### **Example 2**
- Input: `List: 1 -> 3 -> 2 -> 1`
- Output: `-1`
