## Check if a Linked List is Circular Linked List
### Difficulty: _`Easy`_
---
### **Statement**
Given the `head` of a singly linked list, the task is to find if given linked list is a circular or not. A linked list is called `circular` if its last node points back to its first node.

**Note:** The linked list does not contain any internal loop.  



---
### **Example 1**
- Input: `List: 2 -> 4 -> 6 -> 7 -> 5` &circlearrowleft; `2` 
- Output: `True`

### **Example 2**
- Input: `List: 1 -> 2 -> 3 -> 4 -> 5`
- Output: `False`
