## Segregate Evens And Odds In a Linked List
### Difficulty: _`Medium`_
---
### **Statement**
Given a linked list, modify the list such that all the even numbers appear before all the odd numbers in the modified list. The order of appearance of numbers within each segregation should be the same as that in the original list.<br>
**Note:** Don't create a new linked list, instead reaarange/modify the provided one.

---
### **Example 1**
- Input: `List: 17 -> 15 -> 8 -> 9 -> 2 -> 4 -> 6`
- Output: `8 -> 2 -> 4 -> 6 -> 17 -> 15 -> 9`

### **Example 2**
- Input: `1 -> 3 -> 5 -> 7`
- Output: `1 -> 3 -> 5 -> 7`
