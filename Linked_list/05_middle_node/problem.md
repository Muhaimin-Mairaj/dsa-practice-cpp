## Middle Node in a Linked List
### Difficulty: _`Easy`_
---
### **Statement**
Given the `head` of singly linked list, find middle node of the linked list.
- If the number of nodes is `odd`, return the middle node.
- If the number of nodes is `even`, there are two middles nodes, so return the second middle node.


---
### **Example 1**
- Input: `List: 1 -> 2 -> 3 -> 4 -> 5`
- Output: `3`

### **Example 2**
- Input: `List: 10 -> 20 -> 30 -> 40 -> 50 -> 60`
- Output: `40`
