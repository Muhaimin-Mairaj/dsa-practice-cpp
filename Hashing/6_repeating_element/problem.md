## Only Repeating From 1 To N-1
### Difficulty: _`Easy`_
---
### **Statement**
Given an array `arr[]` of size `n` filled with numbers from `1` to `n-1` in random order. The array has only `one repeating element`. The task is to find the repetitive element.

---
### **Example 1**
- Input: `arr = [1, 3, 2, 3, 4]` 
- Output: `3`

### **Example 2**
- Input: `arr = [1, 5, 1, 2, 3, 4]`
- Output: `1`
