## Duplicates Within K Distance in an Array
### Difficulty: _`Easy`_
---
### **Statement**
Given an integer array `arr[]` and an integer `k`, detremine whether there exist two indices `i` and `j` such that `arr[i] == arr[j]` and `|i - j| <= k`. If such a pair exists, return `Yes` otherwise return `No`.  

---
### **Example 1**
- Input: `arr = [1, 2, 3, 4, 1, 2, 3, 4], k = 3` 
- Output: `No`

### **Example 2**
- Input: `arr = [1, 2, 3, 1, 4, 5], k = 3`
- Output: `Yes`
