## Count Maximum Points On The Same Line
### Difficulty: _`Easy`_
---
### **Statement**
Given `n` points on a 2D plane as pair of `(x, y)` co-ordinates, we need to find maximum number of points which lie on the same line.

---
### **Example 1**
- Input: `arr = [{-1, 1}, {0, 0}, {1, 1}, {2, 2}, {3, 3}, {3, 4}]`
- Output: `4`
- Explanation: `The maximum number of points which lie on the same line is 4, those points are {0, 0}, {1, 1}, {2, 2}, {3, 3}`
