# _Leetcode Problem #90_
## Difficulty: _`Medium`_

### **Statement:** 
- Given an integer array _`nums`_ that may contain duplicate elements, return all possible _`subsets`_ (the power set).

- The solution set must not contain duplicate subsets. 

- Return the solution in any order.

---
### **Example 1**
- Input: `nums = [1,2,2]`
- Output: `[[], [1], [1,2], [1,2,2], [2], [2,2]]`

### **Example 2**
- Input: `nums = [0]`
- Output: `[[], [0]]`

---
### **Constraints:**
- `1 <= nums.length <= 10`
- `-10 <= nums[i] <= 10`
