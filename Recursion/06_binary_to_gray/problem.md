## Binary to Gray code using recursion
### Difficulty: _`Medium`_
---
### **Statement**
- Given the Binary code of number as a decimal number, we need to convert this into its equivalent `Gray Code`. Assume that the binary number is in the range of integers. 

- `Note:` In Gray code, two consecutive numbers differ by exactly one bit. The most significant bit `(MSB)` of the Gray code is always the same as the `MSB` of the binary number. 

---
### **Example 1**
- Input: `1001`
- Output: `1101`
- Explanation: `The Gray code for binary 1001 is 1101`

### **Example 2**
- Input: `11`
- Output: `10`
- Explanation: `The Gray Code for binary 11 is 10`
