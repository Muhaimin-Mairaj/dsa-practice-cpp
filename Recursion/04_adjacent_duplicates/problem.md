## Recursively remove all adjacent duplicates
### Difficulty: _`Medium`_
---
### **Statement**
- Given a string `s`, remove all its adjacent duplicates characters recursively

---
### **Example 1**
- Input: `s = mississippi`
- Output: `m`
- Explanation: `mi(ss)i(ss)i(pp)i -> m(iiii) -> m`

### **Example 2**
- Input: `s = abccbccba`
- Output: `""`
- Explanation: `ab(cc)b(cc)ba -> a(bbb)a -> (aa) -> ""`
