## All Possible Paths in a Matrix 
### Difficulty: _`Medium`_
---
### **Statement**
- Given a 2D matrix of dimension `m x n`, the task is to print all the possible paths from `top left` corner to the `bottom right` corner in a 2D matrix with the constraints that from each cell you can either move to right or down only. 

---
### **Example 1**
- Input: `[[1,2,3], [4,5,6]]`
- Output:    
             `1 2 3 6`<br>
             `1 2 5 6`<br>
             `1 4 5 6`<br>

### **Example 2**
- Input: `[[1,2], [3,4]]`
- Output:    
             `1 2 4`<br>
             `1 3 4`<br>
