## How to use

- Clone the repository: ```git clone https://github.com/Muhaimin-Mairaj/dsa-practice-cpp.git```
- Navigate to desired folder: ``` (eg: cd Recursion/01_all_subsets)```
- Compile the code: ```g++ solution.cpp -o solution```
- Run the executable:
    - For Windows: `.\solution.exe`
    - For MacOS/Linux: `./solution`
